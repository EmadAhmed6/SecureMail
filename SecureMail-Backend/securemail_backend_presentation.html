The Security Engine

The 13-Stage Pipeline

01

Email Parsing

Deep-dive decomposition of raw SMTP payloads. Reconstructs complex nested MIME structures, normalizes character encodings, and strips obfuscated hidden layers to prepare a clean data model for downstream analysis.

RFC 5322 Metadata Extraction UTF-8 Transcoding & Normalization Multipart MIME Reconstruction
02

Authentication

Verify the sender's identity through the cryptographic trust triangle (SPF, DKIM, DMARC). Integrated ARC support validates the authentication chain for forwarded mail, ensuring path integrity across the internet.

SPF Path & IPv6 Validation RSA-2048 DKIM Sig Check DMARC Policy Enforcement
03

Reputation

Real-time cross-referencing against massive global intelligence datasets. Queries 40+ specialized feeds including VirusTotal and AbuseIPDB to assign instant risk profiles to every sender IP and domain found in the envelope.

VirusTotal & Spamhaus Query AbuseIPDB Reputation API GreyNoise Scanner Detection
04

Behavioral

Historical pattern profiling through advanced statistical analysis. Identifies deviations in communication frequency, typical time-of-day activity, and sudden shifts in discourse topics using Z-Score calculation against baselines.

Statistical Anomaly Engine Communication Recency Profiling Z-Score Deviation Logic
05

URL Analysis

Recursive evaluation of embedded hyperlinks. Performs redirection expansion, domain homoglyph detection, and real-time category scanning to identify phishing proxies and credential harvesting landing pages hidden behind shorteners.

PhishTank & URLScan Integration Recursive Redirect Expansion Homoglyph / Punycode Check
06

Malware Core

High-speed binary introspection using a gRPC-powered malware engine. Executes heuristic matching and hash-based lookups in a tight 5s synchronous window, with background deep-scanning for zero-day threat detection.

gRPC Malware Scanner Heuristic Static Analysis Sync-Window Adaptive Deadline
07

Rule Engine

Precision-engineered threat detection using a registry of 23+ specialized heuristics. Identifies BEC markers, urgent phishing cues, and advanced HTML obfuscation techniques by evaluating the entire message context.

23+ Native Backend Rules Social Engineering Heuristics BEC & Urgency Detection
08

Graph Map

Entity relationship mapping and risk amplification. Links independent data points into a connected graph to reveal hidden clusters of suspicious intent that single-point analysis might overlook during standard processing.

Entity Co-occurrence Map Graph-based Risk Propagation Relationship Cluster Logic
09

Correlation

Cross-stage signal corroboration and evidence consolidation. Orchestrates disparate detection layers to validate high-confidence attack patterns like 'Anomalous Reply-To' combined with 'Financial Phishing Urgency'.

Multi-Factor Pattern Match Signal Evidence Consolidation Co-occurrence Weighting
10

Risk Scoring

Deterministic risk quantification via a 3-tiered scoring matrix. Balances 'Hard Signals' (Auth/Rep) with 'Soft Signals' (Patterns/Behavior) and 'Amplifiers' to yield a precise final threat index between 0 and 100.

Tier 1-3 Weighted Matrix Confidence-Adjusted Sum Dynamic Risk Thresholding
11

Verdict Logic

Final policy orchestration based on the calculated risk surface. Maps the numeric score to an actionable state (SAFE, SPAM, PHISHING, MALICIOUS) and determines the final delivery outcome and mailbox folder.

Categorical Label Mapping Folder Routing Logic Outcome Decision Matrix
12

AI Synthesis

LLM-powered reasoning for complex threat interpretation. Generates a comprehensive, human-readable security report, explaining the reasoning behind the verdict and providing actionable remediation steps for the user.

LLM Reasoning Backend Human-Readable Insights Actionable Remediation Guide
13

Persistence & Push

Atomic transaction lifecycle management. Commits the full security context to PostgreSQL via Prisma and emits real-time events through high-speed WebSockets for immediate monitoring and SOC visibility.

Prisma Atomic Transactions WebSocket Push Notification Real-time Event Emission
